## 📂 Task-03: Text Generation with Markov Chains

### Description
An implementation of a statistical text generation algorithm using **Markov Chains**. Unlike deep learning models, this approach predicts the probability of the next word or character based strictly on the transition probabilities found in the provided training text.

### Features
 * **Dynamic Training:** Users can input custom training text directly into the GUI.
 * **Adjustable Parameters:** Control the "Start Word" and the total "Number of Words" to be generated.
 * **Efficient Processing:** Lightweight and fast generation without the need for a GPU.
   
### Tech Stack
 * Python
 * NumPy
 * Tkinter (for the GUI)
